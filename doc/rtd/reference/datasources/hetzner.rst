Hetzner
=======

The ``Hetzner`` datasource retrieves instance metadata and user data from
Hetzner's metadata service.

This datasource is typically used on virtual machines running in the
Hetzner Cloud environment.

**dsname:** ``Hetzner`` (see :doc:`../datasource_dsname_map`)  
**Module:** ``DataSourceHetzner``
