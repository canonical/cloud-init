Bigstep
=======

The ``Bigstep`` datasource retrieves instance metadata and user data from
Bigstep's metadata service.

This datasource is typically used on virtual machines provided by the
Bigstep cloud platform.

**dsname:** ``Bigstep`` (see :doc:`../datasource_dsname_map`)  
**Module:** ``DataSourceBigstep``
