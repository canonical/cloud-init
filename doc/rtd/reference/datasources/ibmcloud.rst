IBM Cloud
=========

The ``IBMCloud`` datasource retrieves instance metadata and user data from
IBM Cloud's metadata service.

This datasource is typically used on virtual machines running in IBM Cloud
environments that support cloud-init.

**dsname:** ``IBMCloud`` (see :doc:`../datasource_dsname_map`)  
**Module:** ``DataSourceIBMCloud``
